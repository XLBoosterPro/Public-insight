GitHub Setup Guide

GitHub Username (Owner)Your 'Owner' name is simply your GitHub Username.
Example: If your profile link is github.com/john-doe, then Username is john-doe.
2
Create Repository (Repo Name)Click '+' (Top Right) on GitHub > 'New repository'. Enter a name (e.g., 'my-dashboard').
IMPORTANT Select PUBLIC and check Add a README file. Click Create.
3
Branch NameBy default, GitHub creates a branch named main. It is already pre-filled. Do not change it.
4
Create Live Link (Base URL)In your Repo: Go to Settings > Pages (left sidebar). Under Build/Deployment, select Branch: main and click SAVE. Wait 1 min, refresh, and copy the link 'Your site is live at https://...'

GitHub Token (Access Password) Click here to open Token Page

Create Custom Calculated Field

2 Write Formula (Excel Style)
Rule 1: Always start with an = (equals sign).   Rule 2: Type [ to easily pick columns.
Quick Reference Guide
Basic Math (+, -, *, /)= [Revenue] - [Cost]Calculates difference between two numeric columns.
IF Condition= IF([Age] >= 18, "Adult", "Kid")Check logic. Works with 'if' or 'IF'. Use quotes for text.
Multi IF (Nested)= IF([Score]>90, "A", IF([Score]>50, "B", "C"))Chain multiple conditions easily.
AND / OR Logic= IF(AND([Qty]>10, [Price]>100), "Yes", "No")Combine multiple rules into one single check.
Documentation: Custom Fields
Level 1 Beginner: Basic Math
Use standard mathematical operators (+, -, *, /) to create new fields. Remember, columns must be wrapped in square brackets [ ].
= [Selling Price] - [Buying Price] = [Total Sales] / [Quantity]
Level 2 Intermediate: Logical Functions (IF, AND, OR)
You can use conditions to categorize data automatically. If returning text, always use double quotes "".
Single IF: IF(Condition, True_Value, False_Value)
= IF([Profit] > 0, "Profitable", "Loss")
Multiple IFs (Nested):
= IF([Age] < 18, "Kid", IF([Age] < 60, "Adult", "Senior"))
AND / OR (Multiple Conditions at once):
= IF(AND([Sales] > 500, [Rating] == 5), "Top Product", "Normal")
Level 3 Advanced: Text & Variables
Because this engine runs on JavaScript, you can use native JS text manipulations.
Combine Text (Concatenate):
= [First Name] + " " +[Last Name]
Make Uppercase / Lowercase:
= [City].toUpperCase()
Golden Rules to remember
1. Always start your formula with an = sign.
2. Column names must be typed exactly as they appear, or just type [ and select from the popup.
3. This tool is case-insensitive for IF, AND, OR. (i.e., 'if' and 'IF' both work).
